# Library Tracker
Hello everyone! Welcome to my Library Tracker Web app!

If you would like to see the project at anytime it is launched in gitpages to see it functioning or heck if you wanna use it awesome!

## About
This project was inspired by my wife. She really wanted a way to track her books. Now yeah she can use excel or continue using her google docs but as a programmer... I had to make something cool.

My end goal is to turn this into an app that she can access everywhere, phone, tablet, pc, you name it. 

## Functions
The tracker in its current state can record:
* Author
* Book Title
* Cover Image
  * Must use a url for this
* Date finished
* Rating up to 5 stars
* Flair for book being finished or unfinished

### How it works
Once your data is filled out from the form, it is saved into your browsers local data. So as long as you continue to use the pages link and the same browser you will always have the data... just dont reset your browser data. 

If your wanting to move your data around or maybe even save your progress youll have to download the local browser data for the site in the dev tools. This will also work if you change your pc or what not.

Eventually if its ever hosted their will be back-end servers to save data but this works perfect for the type of project.
